# Project NWorld

> **AI가 당연해진 세상에서, 다음에 올 "상상만 했던 미래"를 찾고 만든다.**

## Mission

다가올 미래에 사람들의 행동과 문화를 바꿀 혁신적인 제품/서비스를 설계하고 구축한다.
AI 시대의 전환점에서 패러다임 전환을 리드하는 것이 목표다.

## Current Phase

**Discovery & Research** (2026년 2월~)

가설 → 검증 → 학습 사이클을 반복하며 "AI의 교외(suburb)"를 찾는 중.

---

## Core Insight

4개의 독립적 리서치 스트림(역사적 창발 패턴, 기술 혁명의 그림자, Post-AI 신호, 철학적 본질)을 종합한 결과:

> **모든 기술 혁명의 진짜 변화는 기술 자체가 아니라, 기술이 만든 "새로운 당연함" 위에서 피어난 "상상도 못한 것"이었다.**

| 기술 | 사람들이 생각한 혁명 | 진짜 혁명 |
|------|------------------|----------|
| 자동차 | 빠른 이동 | 교외, 패스트푸드, 모텔, 데이트 문화 |
| 전기 | 더 밝은 불 | 냉장고, 세탁기 → 여성 사회진출 |
| 인터넷 | 전자 도서관 | 소셜 네트워크, 공유 경제, 인플루언서 |
| 스마트폰 | 더 좋은 전화기 | 긱 이코노미, 셀피 문화, 배달앱 |
| **AI** | **더 빠른 검색/코딩** | **???** |

지금 AI는 역사가 반복적으로 보여준 **"말 없는 마차(horseless carriage)" 단계** — 새 기술을 구 패러다임에 끼워 넣는 시기. 진짜 혁명은 이 단계가 끝난 후에 온다.

## Key Findings

### 7가지 역사적 창발 패턴
1. **30년 법칙** — 기술→변혁 간격이 가속 (77년→40년→10년→5년→**AI: 2-5년?**)
2. **구 패러다임 덧씌우기** — 전기를 증기벨트에 연결하던 시대 (Paul David)
3. **매체가 메시지** — 기술의 형식/구조가 사회를 바꾼다 (McLuhan)
4. **의도의 역전** — 발명가 의도와 정반대 결과가 반복됨
5. **설치→광란→붕괴→황금기** — AI는 지금 "광란기" (Carlota Perez)
6. **새로운 당연함의 적층** — AI가 만들 "당연함 6"은?
7. **연결의 역설** — 모든 연결은 새로운 단절을 만든다

### 5가지 그림자 법칙
1. **그림자 산업** — 기술 부작용 해결이 기술 자체보다 더 큰 산업이 된다
2. **Transition Tax 감소 = 기회** — 전환 비용을 줄여주는 자가 승리
3. **2차 효과 시차** — 문화 변화는 10-30년 뒤에 발현
4. **라가드 = 진짜 대중 시장** — 인구의 50%가 늦게 채택하는 순간이 폭발
5. **저항 = 기회의 신호** — 가장 강한 저항이 가장 큰 시장

### 7가지 철학적 성공 원칙
1. "유용한 지식"에 대한 문화적 합의 (Bacon-Mokyr)
2. 혁신가에 대한 사회적 존경 (McCloskey)
3. 아이디어의 자유 시장 (Mokyr의 "학술 공화국")
4. 제도적 적응 능력 (Perez의 "전환점")
5. 파괴와 보상의 균형 (Schumpeter)
6. 오래된 권위의 해체
7. 실험적 사고방식의 문화적 수용 (실용주의)

### 핵심 질문 (탐색 중)

> **"AI의 교외(suburb)는 무엇인가?"**
>
> 자동차의 킬러 앱은 자동차가 아니라 교외였다.
> 전기의 킬러 앱은 전구가 아니라 냉장고였다.
> AI의 킬러 앱은 챗봇이 아닐 것이다. 그러면 무엇인가?
