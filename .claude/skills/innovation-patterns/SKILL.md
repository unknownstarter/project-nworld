# Innovation Patterns

## 파괴적 혁신의 7가지 패턴

### 1. 10x Convenience (10배 편의성)
기존 행동을 10배 더 쉽게 만든다.
- Google: 도서관 검색 → 타이핑 한 번
- Uber: 택시 잡기 → 버튼 한 번
- ChatGPT: 여러 페이지 읽기 → 직접 답변
- **질문**: 사람들이 매일 하지만 불필요하게 복잡한 것은?

### 2. Democratize Access (접근성 민주화)
전문가만 할 수 있던 것을 누구나 할 수 있게.
- WordPress: 웹 개발 → 누구나 사이트 제작
- Canva: 디자인 → 드래그 앤 드롭
- 바이브코딩: 프로그래밍 → 자연어로 설명
- **질문**: 전문가만 할 수 있어서 대부분의 사람이 포기하는 것은?

### 3. Remove Middlemen (중개인 제거)
불필요한 중간 단계를 제거한다.
- Amazon: 소매점 → 직배송
- Airbnb: 호텔 → 직접 숙소 연결
- AI Agent Commerce: 검색+비교+구매 → 에이전트가 한번에
- **질문**: 사용자와 가치 사이에 불필요한 중간 단계는?

### 4. Aggregate then Curate (모으고 큐레이션)
흩어진 것을 모은 후 개인화하여 제공.
- Netflix: 비디오 대여점 → 무제한 스트리밍 + 추천
- Spotify: CD 가게 → 모든 음악 + Discover Weekly
- TikTok: 콘텐츠 바다 → AI 큐레이션
- **질문**: 좋은 것이 많지만 찾기 어려운 영역은?

### 5. Create New Habit Loops (새로운 습관 루프)
Hook 모델: Trigger → Action → Variable Reward → Investment
- Instagram: 알림 → 열기 → 새 콘텐츠 → 좋아요/팔로우
- Duolingo: 리마인더 → 퀴즈 → 스트릭 → 계속 학습
- **질문**: 사람들이 매일 반복할 수 있는 작은 보상 루프는?

### 6. Platform Play (플랫폼 전략)
양면 시장을 만들어 네트워크 효과를 발생시킨다.
- App Store: 개발자 ↔ 사용자
- YouTube: 크리에이터 ↔ 시청자
- Shopify: 판매자 ↔ 인프라
- **질문**: 생산자와 소비자를 더 잘 연결할 수 있는 시장은?

### 7. Unbundle then Rebundle (해체 후 재결합)
큰 제품에서 하나를 떼어내 더 잘 만들고, 다시 확장.
- Craigslist 해체: Airbnb(숙소), Indeed(채용), Tinder(만남)
- Excel 해체: Airtable(DB), Notion(문서), Figma(디자인)
- Google 해체: Perplexity(답변), ChatGPT(대화), Midjourney(이미지)
- **질문**: 기존의 거대 서비스에서 떼어내 10배 잘할 수 있는 부분은?

## 창의적 사고 기법

### First Principles Thinking
```
현상: "사람들이 X를 하려면 Y가 필요하다"
질문: "정말 Y가 필수적인가? 근본적으로 무엇이 필요한 것인가?"
제거: Y 없이 X를 할 수 있는 방법은?
```

### Inversion
```
정방향: "어떻게 성공할까?"
역방향: "어떻게 하면 반드시 실패할까?"
→ 실패 조건을 피하면 성공 확률 증가
```

### Analogy Transfer
```
다른 산업의 성공 패턴을 현재 문제에 적용:
"Uber for X", "Airbnb for Y"
→ 더 깊이: 어떤 메커니즘이 작동했는가?
```

### What If...?
```
제약 조건을 제거하고 상상:
- 비용이 0이라면?
- 시간이 무한하다면?
- 100배 스케일이라면?
- 반대로 한다면?
```
