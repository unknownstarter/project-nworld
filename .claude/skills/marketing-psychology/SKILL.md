# Marketing Psychology & Behavioral Science

출처: coreyhaines31/marketingskills (marketing-psychology) + 행동경제학 원리

## Core Principles

### 1. Loss Aversion (손실 회피)
사람은 이득보다 손실에 2배 민감하다.
- "이 기능을 놓치지 마세요" > "이 기능을 사용해보세요"
- 무료 체험 종료 알림: "저장한 데이터가 삭제됩니다"
- 카운트다운 타이머: 긴급성 조성

### 2. Social Proof (사회적 증거)
다른 사람이 하면 따라한다.
- "10만 명이 사용 중"
- 실시간 사용자 수 표시
- 고객 리뷰, 케이스 스터디
- 유명 기업 로고

### 3. Anchoring (닻 내리기)
처음 본 정보가 판단 기준이 된다.
- 가격: 비싼 플랜을 먼저 보여주면 저렴한 플랜이 합리적으로 보임
- "정가 $99 → 할인가 $49"
- 경쟁사 가격 대비 포지셔닝

### 4. Scarcity (희소성)
제한되면 더 원한다.
- "남은 자리 3개"
- "이번 주까지만 이 가격"
- 초대제 / 대기자 명단
- IMPORTANT: 거짓 희소성은 신뢰를 파괴한다

### 5. Reciprocity (상호성)
받으면 돌려주고 싶다.
- 무료 가치 제공 (블로그, 도구, 템플릿) → 유료 전환
- Free tier의 넉넉한 제공
- 개인화된 추천/인사이트

### 6. Default Effect (기본값 효과)
대부분의 사람은 기본 설정을 바꾸지 않는다.
- 기본 플랜을 권장 플랜으로
- 옵트아웃(기본 참여) vs 옵트인(기본 미참여)
- 가장 원하는 옵션을 기본값으로

### 7. Endowment Effect (소유 효과)
가진 것에 더 높은 가치를 부여한다.
- 무료 체험에서 데이터/설정 축적 → 이탈 비용 증가
- "내 프로필", "내 워크스페이스"
- 커스터마이징 유도

## 행동 변화 모델 (B=MAT, Fogg)
```
Behavior = Motivation × Ability × Trigger
```
- **Motivation**: 이 행동을 왜 해야 하는가? (가치 제안)
- **Ability**: 얼마나 쉬운가? (마찰 제거)
- **Trigger**: 언제 행동하도록 유도하는가? (알림, CTA)

3가지 중 하나라도 0이면 행동은 일어나지 않는다.
→ 전환율이 낮으면: M, A, T 중 무엇이 부족한지 진단

## 인지 편향 활용 체크리스트

- [ ] Choice Architecture: 선택지를 최적으로 배열했는가?
- [ ] Progress Illusion: 진행감을 느끼게 했는가? (체크리스트, 프로그레스 바)
- [ ] Peak-End Rule: 마지막 경험이 긍정적인가?
- [ ] Mere Exposure: 반복 노출로 친숙함을 만들었는가?
