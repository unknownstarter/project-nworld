# Conversion Rate Optimization (CRO)

출처: coreyhaines31/marketingskills (page-cro, signup-flow-cro, onboarding-cro, form-cro, popup-cro, paywall-upgrade-cro)

## CRO Framework: R.E.S.E.A.R.C.H.

### 1. Record (관찰)
- Heatmap 분석 (Hotjar, PostHog)
- Session recording 검토
- Scroll depth 측정
- Click map 분석

### 2. Evaluate (평가)
현재 전환 퍼널의 각 단계별 이탈률:
```
방문 → [X% 이탈] → 가입 페이지 → [Y% 이탈] → 활성화 → [Z% 이탈] → 유료 전환
```

### 3. Survey (설문)
- 이탈 시점에서 "왜 떠나시나요?" 마이크로 설문
- NPS/CSAT 조사
- 사용자 인터뷰

## 페이지별 CRO

### Landing Page
- Hero: 명확한 가치 제안 (3초 테스트)
- CTA: 하나의 명확한 행동 유도 (색상 대비, 위치)
- Social Proof: 고객 수, 로고, 추천사
- 불안 해소: FAQ, 보증, 무료 체험

### Signup Flow
- 단계 최소화 (3단계 이하)
- Social login 제공 (Google, GitHub)
- 진행 표시줄
- 이메일 유효성 실시간 체크

### Onboarding
- Time-to-Value 최소화 (첫 성공 경험까지)
- 체크리스트 패턴 (완료감)
- 스킵 가능한 튜토리얼
- 샘플 데이터로 빈 상태 방지

### Paywall / Upgrade
- 무료에서 유료로의 자연스러운 전환점 (기능 제한 도달 시)
- 가격 앵커링 (비싼 플랜을 먼저)
- 연간 결제 할인 강조
- 무료 체험 / 머니백 보장

### Forms
- 필드 최소화 (이름 + 이메일만)
- 인라인 유효성 검사
- 자동완성 지원
- 에러 메시지는 필드 옆에

### Popups
- 타이밍: 종료 의도(exit intent) 또는 스크롤 50%+
- 가치 제안 명확 (뉴스레터 구독 → "주간 인사이트 받기")
- 닫기 버튼 명확하게
- 세션당 1회만

## A/B Test Priorities
Impact × Confidence ÷ Effort = 우선순위
1. CTA 카피/색상 (Quick Win)
2. 가입 플로우 단계 수
3. 가격 페이지 레이아웃
4. 온보딩 흐름
