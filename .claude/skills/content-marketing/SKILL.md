## Copywriting Frameworks

### PAS (Problem-Agitation-Solution)
```
Problem: [문제 공감]
Agitation: [문제를 안 풀면 생기는 결과]
Solution: [우리 제품이 해결]
```

### AIDA (Attention-Interest-Desire-Action)
```
Attention: 눈을 사로잡는 헤드라인
Interest: "나와 관련있다" 느끼게
Desire: "갖고 싶다" 느끼게
Action: 명확한 CTA
```

### Before-After-Bridge
```
Before: [현재 고통스러운 상태]
After: [이상적인 미래 상태]
Bridge: [우리 제품이 그 다리]
```

## Email Sequence Patterns

### Welcome Sequence (5-7통)
```
Day 0: 환영 + 핵심 가치 전달
Day 1: 첫 성공 경험 가이드
Day 3: 사용 사례 / 케이스 스터디
Day 5: 고급 기능 소개
Day 7: 소셜 프루프 + 업그레이드 제안
```

### Nurture Sequence
- 주 1회, 가치 제공 (80%) + 판매 (20%)
- 개인화: 행동 기반 세그멘트
- 제목줄: 40자 이내, 호기심 유발

### Re-engagement
- 30일 미접속 사용자
- "그동안 이런 게 새로 나왔어요"
- 할인/혜택 제공
- 마지막 통보 ("계정이 비활성화됩니다")

## Social Content Strategy

### Platform별 접근
| Platform | 톤 | 포맷 | 빈도 |
|----------|-----|------|------|
| X/Twitter | 날카롭고 간결 | Thread, 스크린샷 | 매일 |
| LinkedIn | 전문적, 인사이트 | 긴 포스트, 캐러셀 | 주 3회 |
| YouTube | 교육적 | 튜토리얼, 데모 | 주 1회 |
| TikTok | 재미있고 빠름 | 짧은 영상 (< 60초) | 매일 |

### Content Pillar Model
3-5개 핵심 주제를 정하고 반복:
1. 제품 업데이트 (10%)
2. 교육 콘텐츠 (40%)
3. 사용자 스토리 (20%)
4. 업계 인사이트 (20%)
5. 비하인드 스토리 (10%)

## Content Strategy Execution
```
1. 키워드 리서치 → 콘텐츠 갭 분석
2. 콘텐츠 캘린더 작성 (월별)
3. 재사용: 1 블로그 → Thread + LinkedIn + 뉴스레터
4. 측정: Traffic, Engagement, Conversion per content
5. 반복: Top performing 유형에 집중
```
