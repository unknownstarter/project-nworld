# Competitive Analysis

## Framework

### 1. 경쟁 환경 매핑
- **직접 경쟁**: 같은 문제를 같은 방식으로 풀는 제품
- **간접 경쟁**: 같은 문제를 다른 방식으로 풀는 제품
- **대안**: 사용자가 현재 사용하는 방법 (엑셀, 수작업 포함)

### 2. 경쟁사별 분석
각 경쟁사에 대해:
- 제품: 핵심 기능, UX, 기술 스택
- 비즈니스: 가격 모델, 타겟 고객, 시장 규모
- 전략: 포지셔닝, 메시징, 성장 전략
- 강점/약점: SWOT
- 사용자 리뷰: G2, ProductHunt, App Store 분석

### 3. 비교 매트릭스
```
| Feature          | Us | Comp A | Comp B | Comp C |
|------------------|----|--------|--------|--------|
| [기능 1]         |    |        |        |        |
| [기능 2]         |    |        |        |        |
| Price            |    |        |        |        |
| Target Customer  |    |        |        |        |
| UX Quality       |    |        |        |        |
```

### 4. 차별화 기회
- 경쟁사가 무시하는 세그먼트는?
- 공통적으로 불만인 기능은?
- 기술적으로 가능하지만 아무도 안 하는 것은?
- 비즈니스 모델 혁신 여지는?
